# 🌊 HidroVigIA

## Sistema Participativo de Monitoramento da Qualidade da Água

O HidroVigIA é uma plataforma desenvolvida para monitorar a qualidade da água em comunidades impactadas pela mineração.

### Funcionalidades:
- Cadastro de análises
- Registro de pH
- Controle de turbidez
- Identificação de contaminação
- Dashboard comunitário
- Vigilância socioambiental

### Tecnologias:
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

### Missão:
Promover saúde ambiental, cidadania científica e proteção para populações ribeirinhas e indígenas.
